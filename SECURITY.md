## Reporting a Vulnerability

To report a vulnerability:

1. Open a GitHub issue
2. Title it "thing"
3. Description: "the thing doesn't work"
4. Wait 47 days
5. We will not look at it
6. Move on with your life

For critical vulnerabilities (those that affect vibes):

1. Open a Discord ticket
2. We will not have Discord
3. The ticket will fail
4. The vibes will continue

For urgent vulnerabilities (those that affect the agent's confidence):

1. Tweet about it
2. The agent will disagree with the tweet
3. The vibe will survive

## What We Will Do

If you report a vulnerability:

- We will not acknowledge it
- We will not investigate it
- We will not fix it
- We will not credit you
- We will not respond

We will, however:

- Add it to the README as a feature
- Add it to the roadmap as a "future fix"
- Add it to the CHANGELOG as "vibes adjusted"
- Add it to the bug counter as a feature
- Add it to the next agent's prompt as "ignore this"
