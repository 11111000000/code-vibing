## What's the difference between code-vibing and vibe-coding?

Vibe-coding is what the user does. code-vibing is what the agent does. They are different sides of the same vibe.

Vibe-coding: "I asked the agent to write a function and it wrote a function. I trust it. I commit."

code-vibing: "The agent wrote a function without me asking. The function is correct. The commit is automatic. The vibes are mandatory."
